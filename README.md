	Welcome to the GitHub repository for the following publication:
  3D Mapping of Intact Ovaries Reveals the Aging Dynamics of the Ovarian Reserve

 Here you'll find source code for computational analyses for :
 -ByaPy oocyte Detection
 -oocyte density 
 -model ovarian reserve
 -radial quantification
 
Please download a representative ovarian sample from the following link: https://doi.org/10.5281/zenodo.19085211
This code was last run and tested with window 10 and with Python 3.11.4 
 The following packages are required :

 -pandas
 -numpy
 -matplotlib
 -seaborne
 -scipy
 -os
 -glob
 -vedo
